# Python Projects

This repository contains Python programs that I am building while learning programming from scratch.
It contains a collection of Python projects demonstrating practical programming skills.  
These projects are built to strengthen problem-solving, logical thinking, and hands-on Python experience.

## Projects Overview

1. **Hello World** – Simple program demonstrating Python syntax.
2. **Addition Calculator** – Basic calculator for summing two numbers.
3. **Advanced Calculator** – Enhanced calculator with multiple operations.
4. **To-Do List Manager** – Command-line program to manage tasks efficiently.
5. **Weather App** – Real-time weather information using OpenWeatherMap API.

## Topics Covered
- Variables and data types
- Conditional statements
- Loops
- Functions
- File handling (for To-Do List Manager)  
- Working with APIs (Weather App using OpenWeatherMap)  
- Random number generation (Number Guessing Game)  
- Command-line user interaction  

## Status
Actively learning and adding new programs regularly.

## Tech Stack
- Python 3.x
- Standard Python libraries: `random`, `datetime`, `os`, `requests`

## How to Run
1. Clone the repository:  
   ```bash
   git clone https://github.com/arya-sharma-dev/Python-Projects.git

2. Navigate into the cloned folder:
   cd PythonProjects

3. Enter the folder of the project you want to run:
   cd FolderName

Replace FolderName with the project folder you want (e.g., HelloWorld, ToDoListManager).

4. Run the Python file:
   python FileName.py
Replace FileName.py with the actual Python file in that folder (e.g., HelloWorld.py).
